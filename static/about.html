About This Server

This HTTP server is built from scratch in Rust without using any external crates!

Features:

  • ✅ TCP connection handling
  • ✅ HTTP request parsing
  • ✅ HTTP response generation
  • ✅ Basic routing system
  • ✅ Static file serving
  • ✅ Query parameter parsing
  • ✅ Multiple MIME types
