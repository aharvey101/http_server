Welcome to our HTTP server built from scratch in Rust!

Available Routes:

  • GET /hello - Simple greeting
  • GET /hello?name=Rust - Greeting with query parameter
  • GET /api/status - Server status JSON
  • POST /api/echo - Echo back request data

Test with curl:

curl -X POST http://127.0.0.1:8080/api/echo -d "Hello from curl!"
